MAC Keyboard Shortcuts
May 19th, 2016

How to get to a hidden folder from the finder:

  1. Open finder
  2. shift + CMD + g
  3. Type in the folder path

Switch between applications:

  1. CMD + tab
  2. Hold the option (alt) button before letting go of CMD. This will bring up the app if it was minimized.

Switch between tabs:

  1. CMD + shift + {
  2. CMD + shift + }

Spotlight search:

  1. CMD + space
